NU World Cup Registration

Redesigned Northwestern World Cup's homepage to also include a single page webapp (SPA) to increase the efficiency of the registration process. This webapp is built using the MEAN Stack, using MongoDB, Express.js, Angular.js, and Node.js.

Assembly Language Game Development

This is a 2-D side scrolling game that implements simple physics, collision detection, and fixed point arithmetic. The program is coded for the Intel IA-32 architechture using MASM32 directives. It also implements fundamental optimization techniques including common subexpression elimination and loop unrolling.


Neuroscience Research Automated Testing

Using Arduino (C++) and Matlab, we created scripts to train mice in certain behaviors and then test their innate olfactory system's sensitivities using custom built circuitry.

Algorithmic Stock Analytics

Created a webapp using Ruby on Rails and Yahoo Query Language to analyze historical stock data and provide confidence intervals for long term estimated profitability.
